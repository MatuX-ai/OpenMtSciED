✅ 支持的认证方式

📧
邮箱密码登录
传统的邮箱密码认证方式，支持注册、登录、密码重置等功能
🐙
GitHub OAuth
通过GitHub账号快速登录，支持用户信息获取和权限管理
🔍
Google OAuth
Google账号登录，支持OpenID Connect协议和用户资料同步
💬
微信登录
微信扫码登录，支持snsapi_login和snsapi_userinfo两种授权方式
🐧
QQ登录
QQ快捷登录，获取用户基本信息，符合QQ互联平台规范

💻 使用示例

// 初始化认证服务
import { initAuth } from './auth';

const authService = initAuth({
apiUrl: 'https://api.yourdomain.com',
githubClientId: 'your_github_client_id',
googleClientId: 'your_google_client_id',
wechatAppId: 'your_wechat_app_id',
qqAppId: 'your_qq_app_id'
});

// 微信登录
await authService.signInWithWeChat({
scope: 'snsapi_login',
redirectUri: 'https://yourdomain.com/auth/callback'
});

// QQ登录
await authService.signInWithQQ({
scope: 'get_user_info',
redirectUri: 'https://yourdomain.com/auth/callback'
});
