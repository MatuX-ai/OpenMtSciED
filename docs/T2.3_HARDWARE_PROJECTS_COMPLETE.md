## 📋 任务概述

T2.3 硬件项目管理旨在为 OpenMTSciEd Desktop Manager 添加低成本STEM硬件项目支持，所有项目预算≤50元，适合普惠STEM教育。该功能包括数据模型设计、项目列表展示和 Blockly 可视化编程集成。

## ✅ 已完成工作

### T2.3.1 硬件项目数据模型 (0.15人天)

**文件:** `desktop-manager/src/app/models/hardware-project.models.ts`

**实现内容:**
1. **核心接口定义**
   - `HardwareProject`: 硬件项目主接口，包含名称、描述、分类、难度、预算等
   - `MaterialItem`: 材料清单项，包含名称、数量、单价、购买链接等
   - `CodeTemplate`: 代码模板，支持 Arduino/MicroPython/Blockly/Scratch
   - `PinConfiguration`: 引脚配置说明

2. **类型定义**
   - `HardwareCategory`: 7种硬件分类（电子电路、机器人、物联网、机械结构、智能家居、传感器应用、通信技术）
   - `CodeLanguage`: 4种编程语言（Arduino C++、MicroPython、Blockly、Scratch）
   - `HardwareProjectFilter`: 筛选条件接口

3. **辅助函数**
   - `calculateTotalCost()`: 计算材料总成本
   - `validateBudget()`: 验证项目预算是否符合≤50元要求
   - `getDifficultyStars()`: 获取难度星级显示
   - `getCategoryName()`: 获取分类中文名称
   - `getCodeLanguageName()`: 获取编程语言显示名称

**特点:**
- ✅ 完整的 TypeScript 类型定义
- ✅ 严格的预算验证机制（≤50元）
- ✅ 丰富的辅助工具函数
- ✅ 支持多种编程语言和硬件分类

## 📝 关键技术决策

### 1. 为什么选择 Blockly？
- **可视化编程**: 降低编程门槛，适合初学者
- **代码生成**: 自动生成 Arduino/Python 代码
- **生态丰富**: Google 维护，社区活跃
- **可扩展性**: 支持自定义积木块

### 2. 为什么限制预算≤50元？
- **普惠教育**: 确保所有学生都能负担
- **开源精神**: 符合 OpenMTSciEd 的开放理念
- **实用性**: 50元足以完成大多数基础项目

### 3. 为什么支持多种编程语言？
- **灵活性**: 适应不同学习阶段
- **渐进式学习**: 从 Blockly → Arduino → Python
- **兼容性**: 支持多种硬件平台
