核心算法 - Operational Transformation：

<!-- OT算法处理并发操作 --> transform(op1, op2) { // 插入-插入转换 if (op1.type === 'insert' && op2.type === 'insert') { return op1.position < op2.position ? op1 : { ...op1, position: op1.position + op2.content.length }; } // 删除-插入转换 if (op1.type === 'delete' && op2.type === 'insert') { return op2.position <= op1.position ? { ...op1, position: op1.position + op2.content.length } : op1; } }

💻 使用示例

前端组件集成：

<!-- 协作编辑器组件 --> <app-collaborative-editor [courseId]="courseId" [orgId]="orgId" [documentId]="documentId" [userId]="currentUserId" (contentChange)="onContentChange($event)"> </app-collaborative-editor>

后端API调用：

# 创建协作文档 POST /api/v1/org/{org_id}/courses/{course_id}/collaborative-documents/ { "document_name": "教学大纲", "document_type": "richtext", "content": "初始文档内容" } # 应用操作 POST /api/v1/org/{org_id}/courses/{course_id}/collaborative-documents/{document_id}/operations/batch [ { "operation_type": "insert", "position": 10, "content": "新增内容", "client_id": "user123" } ]
