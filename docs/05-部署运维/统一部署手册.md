# 统一部署手册

## 概述

本文档涵盖iMatu平台各个模块的完整部署流程，包括基础环境搭建、各服务部署、配置管理和运维监控。

## 系统架构

### 整体部署架构
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   负载均衡层    │    │   应用服务层    │    │   数据存储层    │
│  (Nginx/HAProxy)│    │                 │    │                 │
├─────────────────┤    ├─────────────────┤    ├─────────────────┤
│ - SSL终止       │    │ - 前端应用      │    │ - PostgreSQL    │
│ - 请求分发      │    │ - 后端API       │    │ - Redis缓存     │
│ - 静态文件      │    │ - 区块链网关    │    │ - CouchDB       │
└─────────────────┘    │ - AI服务        │    │ - 文件存储      │
                       │ - 移动端服务    │    └─────────────────┘
                       └─────────────────┘
```

#### 操作系统初始化
```bash
# Ubuntu 22.04 LTS 初始化
sudo apt update && sudo apt upgrade -y
sudo apt install -y docker.io docker-compose git curl wget

# 启动Docker服务
sudo systemctl enable docker
sudo systemctl start docker

# 添加当前用户到docker组
sudo usermod -aG docker $USER
```

### Docker环境配置

#### Docker Compose文件
```yaml
version: '3.8'

services:
  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx/conf:/etc/nginx/conf.d
      - ./nginx/certs:/etc/nginx/certs
    depends_on:
      - frontend
      - backend

  frontend:
    build: 
      context: ./frontend
      dockerfile: Dockerfile.prod
    environment:
      - NODE_ENV=production

  backend:
    build:
      context: ./backend
      dockerfile: Dockerfile.prod
    environment:
      - ENV=production
    depends_on:
      - postgres
      - redis

  postgres:
    image: postgres:15-alpine
    environment:
      POSTGRES_DB: imatu_prod
      POSTGRES_USER: imatu_user
      POSTGRES_PASSWORD: ${DB_PASSWORD}
    volumes:
      - postgres_data:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    command: redis-server --appendonly yes
    volumes:
      - redis_data:/data

volumes:
  postgres_data:
  redis_data:
```

## 各模块部署

### 1. 前端应用部署

#### Angular应用构建
```bash
# 构建生产版本
cd frontend
npm ci
npm run build:prod

# Docker镜像构建
docker build -f Dockerfile.prod -t imatu/frontend:latest .
```

#### Nginx配置
```nginx
server {
    listen 80;
    server_name imatu.example.com;
    
    root /usr/share/nginx/html;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://backend:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 2. 后端服务部署

#### FastAPI应用
```dockerfile
FROM python:3.11-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .
EXPOSE 8000

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```

## 安全配置

### SSL证书配置
```bash
# Let's Encrypt证书申请
sudo certbot --nginx -d imatu.example.com

# 或使用自有证书
sudo cp certificate.crt /etc/nginx/certs/
sudo cp private.key /etc/nginx/certs/
```

### 防火墙配置
```bash
# UFW防火墙规则
sudo ufw allow ssh
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw allow 22/tcp
sudo ufw --force enable
```

### 安全头配置
```nginx
# Nginx安全头
add_header X-Frame-Options "SAMEORIGIN" always;
add_header X-XSS-Protection "1; mode=block" always;
add_header X-Content-Type-Options "nosniff" always;
add_header Referrer-Policy "no-referrer-when-downgrade" always;

```

## 备份恢复

### 数据备份策略
```bash
#!/bin/bash
# backup.sh

DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="/backup/imatu/$DATE"

# 数据库备份
pg_dump -h postgres -U imatu_user imatu_prod > $BACKUP_DIR/database.sql

# 文件存储备份
tar -czf $BACKUP_DIR/files.tar.gz /app/uploads

# 区块链数据备份
docker exec cli peer channel fetch newest /backup/channel.block -c imatu-channel

# 上传到云存储
aws s3 sync $BACKUP_DIR s3://imatu-backups/$DATE/
```

5. 监控系统状态
