## 数据流

### 创建节点流程

```
应用代码
   │
   │  adapter.create_knowledge_node(node_data)
   ▼
ArcadeDBAdapter
   │
   │  构建 Cypher MERGE 查询
   ▼
ArcadeDBClient
   │
   │  POST /api/v1/command/{database}
   │  {
   │    "language": "cypher",
   │    "command": "MERGE (n:KnowledgeNode {...}) SET ...",
   │    "params": {...}
   │  }
   ▼
ArcadeDB Server
   │
   │  Cypher Parser → Query Planner → Execution Engine
   ▼
Storage Engine
   │
   │  写入磁盘（LSM-Tree）
   ▼
返回结果
   │
   │  {"result": [...]}
   ▼
应用代码接收响应
```

### 查询路径流程

```
应用代码
   │
   │  adapter.find_shortest_path(start_id, end_id)
   ▼
ArcadeDBAdapter
   │
   │  构建 Cypher shortestPath 查询
   ▼
ArcadeDBClient
   │
   │  POST /api/v1/command/{database}
   ▼
ArcadeDB Server
   │
   │  图遍历算法（BFS/Dijkstra）
   ▼
返回路径节点列表
   │
   ▼
应用代码处理结果
```

## 迁移策略

### 阶段 1: 并行运行

```
┌─────────────────────────────────────────────┐
│         应用层                               │
│                                             │
│  ┌──────────────────────────────────┐      │
│  │  配置开关: USE_ARCADEDB = False  │      │
│  └──────────────┬───────────────────┘      │
│                 │                           │
│        ┌────────┴────────┐                 │
│        │                 │                 │
│   ┌────▼────┐      ┌────▼────┐           │
│   │ Neo4j   │      │ArcadeDB │           │
│   │ (主)    │      │ (备)    │           │
│   └─────────┘      └─────────┘           │
└─────────────────────────────────────────────┘

特点:
- 默认使用 Neo4j
- ArcadeDB 作为备选
- 双写测试数据
```

### 阶段 2: 逐步迁移

```
┌─────────────────────────────────────────────┐
│         应用层                               │
│                                             │
│  ┌──────────────────────────────────┐      │
│  │  按模块切换:                      │      │
│  │  • 模块A → ArcadeDB ✅           │      │
│  │  • 模块B → Neo4j                 │      │
│  │  • 模块C → Neo4j                 │      │
│  └──────────────────────────────────┘      │
│                                             │
│  逐步验证每个模块的功能和性能                │
└─────────────────────────────────────────────┘

特点:
- 低风险迁移
- 可回滚
- 充分测试
```

### 阶段 3: 完全切换

```
┌─────────────────────────────────────────────┐
│         应用层                               │
│                                             │
│  ┌──────────────────────────────────┐      │
│  │  配置开关: USE_ARCADEDB = True   │      │
│  └──────────────┬───────────────────┘      │
│                 │                           │
│        ┌────────┴────────┐                 │
│        │                 │                 │
│   ┌────▼────┐      ┌────▼────┐           │
│   │ Neo4j   │      │ArcadeDB │           │
│   │ (归档)  │      │ (主)    │           │
│   └─────────┘      └─────────┘           │
└─────────────────────────────────────────────┘

特点:
- ArcadeDB 作为主数据库
- Neo4j 保留用于回滚
- 监控性能和稳定性
```
