## 🔧 技术细节

### 路由守卫

- **AuthGuard**: 保护所有需要认证的路由
  - 未登录用户访问受保护路由时，会被重定向到 `/auth/login`
  - 位置: `desktop-manager/src/app/core/guards/auth.guard.ts`

### 懒加载

所有路由都使用 Angular 的懒加载机制 (`loadComponent`)，按需加载组件，优化首屏加载速度。

### 通配符路由

```typescript
{ path: '**', redirectTo: '/home' }
```

任何未匹配的路径都会重定向到首页。
