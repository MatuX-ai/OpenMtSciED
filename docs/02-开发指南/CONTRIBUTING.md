# Contributing to OpenMTSciEd

Thank you for your interest in contributing! We welcome contributions from the community.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Commit your changes** with clear messages.
3. **Open a Pull Request** describing your changes.

## Development Setup

See [INSTALLATION.md](docs/INSTALLATION.md) for local setup instructions.

## Code Style

- **Python**: Follow PEP 8. Use `black` for formatting.
- **TypeScript/React**: Follow the existing ESLint configuration.

## Reporting Issues

If you find a bug, please open an issue with steps to reproduce it.
