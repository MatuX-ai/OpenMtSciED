🚀 系统功能概览

📧
邮箱密码认证
支持用户注册、登录、密码重置等完整流程，包含邮箱验证功能。
🐙
GitHub OAuth
通过GitHub账号快速登录，获取用户公开信息，支持二次授权。
🔍
Google OAuth
Google账号登录，支持OpenID Connect协议，获取用户资料和邮箱。
💬
微信登录
微信扫码登录，支持snsapi_login和snsapi_userinfo两种授权方式。
🐧
QQ登录
QQ快捷登录，获取用户基本信息，符合QQ互联平台开发规范。
🛡️
安全特性
JWT令牌管理、自动刷新、CSRF防护、State参数验证等安全机制。

💻 代码使用示例

基本初始化

import { initAuth, getAuthService } from './auth'; const authService = initAuth({ apiUrl: 'https://api.yourdomain.com', githubClientId: 'your_github_client_id', googleClientId: 'your_google_client_id', wechatAppId: 'your_wechat_app_id', qqAppId: 'your_qq_app_id' });

微信登录示例

async function loginWithWeChat() { try { await authService.signInWithWeChat({ scope: 'snsapi_login', redirectUri: 'https://yourdomain.com/auth/callback' }); // 登录成功处理 } catch (error) { console.error('微信登录失败:', error); } }

QQ登录示例

async function loginWithQQ() { try { await authService.signInWithQQ({ scope: 'get_user_info', redirectUri: 'https://yourdomain.com/auth/callback' }); // 登录成功处理 } catch (error) { console.error('QQ登录失败:', error); } }

回调处理示例

async function handleOAuthCallback() { const urlParams = new URLSearchParams(window.location.search); const code = urlParams.get('code'); const state = urlParams.get('state'); if (code && state) { try { const authResponse = await authService.handleOAuthCallback(code, state); // 认证成功，跳转到应用主页 window.location.href = '/dashboard'; } catch (error) { // 处理认证失败 } } }
