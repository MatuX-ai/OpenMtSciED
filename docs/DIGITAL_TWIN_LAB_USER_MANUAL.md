# 数字孪生实验室用户手册

## 快速入门

### 系统要求
- 现代浏览器（Chrome、Firefox、Edge最新版本）
- 稳定的网络连接
- 支持WebGL的显卡

### 访问方式
1. 打开浏览器访问系统
2. 登录您的账户
3. 导航到"数字孪生实验室"模块
4. 创建新的实验会话或加入现有会话

## 基本操作

### 1. 创建电路
1. 在右侧"元件库"中选择所需元件
2. 点击元件按钮将其添加到实验室
3. 在3D视图中拖拽元件到合适位置
4. 使用连接工具将元件连接起来

### 2. 控制仿真
- **开始/停止仿真**: 点击"开始仿真"按钮启动实时计算
- **重置电路**: 点击"重置电路"清除所有状态
- **调整参数**: 双击元件可修改阻值、容值等参数

### 3. 监控状态
- **实时数据显示**: 右侧面板显示各元件的电压、电流、功率
- **总览信息**: 顶部显示整体电路的总功率和总电流
- **设备状态**: 显示连接的物理设备状态

## 高级功能

### 多人协作
1. **创建会话**: 点击"新建会话"生成会话链接
2. **邀请协作者**: 分享会话ID给其他用户
3. **实时同步**: 所有参与者的操作会实时同步

### 设备同步
1. **启用IoT同步**: 切换"AWS IoT同步"开关
2. **连接物理设备**: 按照设备说明书进行配对
3. **状态映射**: 系统自动将虚拟元件映射到物理设备

### 数据导出
- 点击"导出数据"按钮
- 选择导出格式（CSV、JSON、PDF）
- 下载实验数据和报告

## 常见元件说明

### 电阻 (Resistor)
- **符号**: ▭
- **参数**: 阻值（欧姆）
- **功能**: 限制电流流动，消耗电能产生热量

### 电容 (Capacitor)
- **符号**: |||
- **参数**: 容值（法拉）
- **功能**: 储存电荷，隔直通交

### 电感 (Inductor)
- **符号**: ⋯
- **参数**: 感值（亨利）
- **功能**: 储存磁场能量，阻碍电流变化

### 电压源 (Voltage Source)
- **符号**: ⚡
- **参数**: 电压值（伏特）
- **功能**: 提供电路所需的电压

## 故障排除

### 问题1: 页面加载缓慢
**解决方案**:
- 检查网络连接
- 清除浏览器缓存
- 尝试刷新页面

### 问题2: 仿真不同步
**解决方案**:
- 检查WebSocket连接状态
- 重新加入会话
- 联系技术支持

### 问题3: 元件无法添加
**解决方案**:
- 确认会话权限
- 检查元件库存
- 重启浏览器

### 问题4: 设备连接失败
**解决方案**:
- 验证AWS IoT配置
- 检查设备证书
- 确认网络防火墙设置
