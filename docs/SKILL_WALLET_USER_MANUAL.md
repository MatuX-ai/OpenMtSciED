# 技能认证区块链钱包用户手册

## 简介

技能认证区块链钱包是一款专为技能认证设计的移动端应用，基于W3C可验证凭证标准和Hyperledger Fabric区块链技术构建。通过本钱包，您可以安全地管理您的技能证书、申请新的认证，并与其他用户分享您的技能资质。

## 主要功能

### 1. 数字身份管理
- 创建和管理去中心化身份(DID)
- 安全存储私钥和公钥对
- 身份信息备份和恢复

### 2. 技能证书管理
- 查看持有的所有技能证书
- 申请新的技能认证
- 验证他人分享的证书
- 创建可验证展示(VP)

### 3. 区块链交互
- 与技能认证区块链网络直接交互
- 实时查询证书状态
- 提交认证申请

## 快速开始

### 安装应用

1. 从应用商店下载"技能认证钱包"
2. 安装并打开应用
3. 阅读并同意用户协议和隐私政策

### 创建钱包

首次使用时需要创建新钱包：

1. 点击"创建新钱包"
2. 设置安全密码（至少8位，包含字母和数字）
3. 备份助记词（务必妥善保管！）
4. 确认助记词顺序
5. 完成钱包创建

⚠️ **重要提醒**: 助记词是恢复钱包的唯一方式，请离线妥善保管，切勿截图或分享给他人。

### 导入现有钱包

如果您已有钱包：

1. 点击"导入钱包"
2. 选择导入方式：
   - 助记词导入
   - 私钥导入
   - 钱包文件导入
3. 输入相应信息
4. 设置新密码
5. 完成导入

## 详细功能使用指南

### 1. 首页概览

首页显示您的核心信息：
- 钱包地址（DID）
- 持有证书数量
- 待处理申请数量
- 最近活动记录

### 2. 证书管理

#### 查看证书
1. 点击底部导航栏"证书"
2. 浏览所有持有的证书
3. 点击证书查看详情：
   - 技能名称和等级
   - 颁发机构
   - 有效期信息
   - 证书状态
   - 验证信息

#### 申请新证书
1. 在证书页面点击"+"号
2. 选择技能类别
3. 填写申请信息：
   - 技能名称
   - 自评等级
   - 相关证明材料（可选）
4. 提交申请
5. 等待审核结果通知

#### 分享证书
1. 在证书详情页面点击"分享"
2. 选择分享方式：
   - 生成二维码
   - 复制证书链接
   - 导出为文件
3. 接收方可扫码或导入验证证书真实性

### 3. 身份管理

#### 查看身份信息
1. 点击底部"我的"
2. 查看DID标识符
3. 查看公钥信息
4. 查看创建时间

#### 备份钱包
1. 在"我的"页面点击"备份钱包"
2. 输入当前密码验证身份
3. 选择备份方式：
   - 生成备份文件
   - 显示助记词
4. 完成备份操作

#### 恢复钱包
1. 卸载重装应用或在新设备上安装
2. 选择"导入钱包"
3. 使用之前的备份信息恢复

### 4. 设置中心

#### 安全设置
- 修改支付密码
- 启用生物识别（指纹/面部识别）
- 设置手势密码
- 管理受信任设备

#### 网络设置
- 切换区块链网络
- 配置节点地址
- 设置网络超时时间

#### 通知设置
- 开启/关闭各类通知
- 自定义通知声音
- 设置免打扰时段

## 高级功能

- 批量验证证书
- 批量删除过期证书

### 数据统计

- 证书获取时间线
- 认证成功率统计
- 活跃度分析

## 安全指南

### 密码安全
- 使用强密码（至少12位，包含大小写字母、数字、特殊字符）
- 定期更换密码
- 不要在多个平台使用相同密码
- 启用双重验证

### 设备安全
- 及时更新操作系统
- 安装正版防病毒软件
- 避免在越狱/root设备上使用
- 定期清理应用缓存

### 网络安全
- 避免使用公共WiFi进行敏感操作
- 确认网站SSL证书有效性
- 警惕钓鱼网站和诈骗信息
- 定期检查账户活动记录

### 隐私保护
- 审慎分享个人信息
- 了解应用权限设置
- 定期检查隐私设置
- 注销不使用的账户

## 常见问题解答

### 钱包相关

**Q: 忘记密码怎么办？**
A: 如果忘记密码且没有备份助记词，将无法恢复钱包。请务必妥善保管助记词。

**Q: 钱包丢失如何恢复？**
A: 使用备份的助记词可以在任何支持的设备上恢复钱包。

**Q: 可以在多台设备上使用同一个钱包吗？**
A: 可以，使用相同的助记词导入即可，但建议只在可信设备上使用。

### 证书相关

**Q: 证书的有效期是多久？**
A: 证书有效期由颁发机构设定，通常为1-3年，到期后需要重新认证。

**Q: 如何验证证书的真实性？**
A: 应用内置验证功能，也可通过区块链浏览器查询证书上链记录。

**Q: 证书可以转让给他人吗？**
A: 技能证书与个人身份绑定，不可转让，但可以分享验证信息。

### 技术问题

**Q: 应用无法连接到网络怎么办？**
A: 检查网络连接，确认防火墙设置，尝试切换网络环境。

**Q: 交易长时间未确认怎么办？**
A: 可能是网络拥堵导致，可耐心等待或联系技术支持。

**Q: 如何反馈bug或建议？**
A: 在"我的"页面找到"帮助与反馈"，详细描述问题并提交。
