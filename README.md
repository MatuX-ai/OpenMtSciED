# OpenMtSciED
1. 构建连贯学习路径：打通课程库（OpenSciEd等K-12现象驱动课程、格物斯坦开源硬件课程、stemcloud.cn全学科课程）与课件库（OpenStax大学/高中教材、《STEM项目式学习(STEM-PBL)科创教育教学标准》、TED-Ed STEM课程）的STEM知识体系，形成“小学兴趣启蒙→初中跨学科实践→高中深度探究→大学专业衔接”的连贯路径。   2. 适配项目定位：结合MatuX“下沉市场普惠教育”特性，强化AI辅助过渡（用LLM解释K-12到大学的知识点跳跃）。  3. AI可操作：让AI编程机器人理解分离的课程库与课件库（解析元数据、构建关联），并能执行“推荐路径、生成联动项目、联动内容”等操作。
