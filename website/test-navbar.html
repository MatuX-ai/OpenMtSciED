🎨 统一导航组件测试

此页面用于测试统一导航组件的各项功能。请尝试以下操作：

✨ 功能特性

  • ✅ 响应式设计 - 调整浏览器窗口大小，查看桌面端和移动端的不同表现
  • ✅ 滚动效果 - 向下滚动页面，观察导航栏背景变化和阴影效果
  • ✅ 活动链接高亮 - 滚动到不同 section，查看对应导航链接的高亮状态
  • ✅ 平滑滚动 - 点击导航链接，体验平滑滚动到目标位置
  • ✅ 移动端菜单 - 在移动设备上点击汉堡菜单，查看侧滑效果
  • ✅ 悬停动画 - 将鼠标悬停在导航链接上，查看下划线动画

⬇️ 向下滚动以测试导航栏滚动效果 ⬇️

🧪 测试清单

  • □ 导航栏固定在页面顶部
  • □ Logo 显示正常且可点击
  • □ 所有导航链接可点击
  • □ 悬停时链接下方出现渐变色下划线
  • □ 滚动超过 50px 后导航栏背景变深
  • □ 当前 section 对应的链接高亮显示
  • □ 点击锚点链接平滑滚动到目标位置
  • □ 移动端（≤768px）显示汉堡菜单
  • □ 点击汉堡菜单展开/收起导航
  • □ 移动端菜单从右侧滑入

⬇️ 继续滚动测试 ⬇️

📖 使用方法

在新页面中使用统一导航组件非常简单：

  • 1. 引入 CSS
    <link rel="stylesheet" href="css/navbar.css">
  • 2. 添加导航 HTML
    参考 components/navbar.html 文件
  • 3. 引入 JavaScript
    <script src="js/navbar.js"></script>
  • 4. 调整页面边距
    body { padding-top: 80px; }

⬆️ 向上滚动查看导航栏变化 ⬆️
