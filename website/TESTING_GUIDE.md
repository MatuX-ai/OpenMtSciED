## 🐛 常见问题排查

### 问题1: 登录后没有返回website
**原因**: desktop-manager的login组件可能没有重定向逻辑

**解决**: 检查 `desktop-manager/src/app/features/auth/login/login.component.ts` 中是否有：
```typescript
const redirectUrl = localStorage.getItem('redirect_after_login');
if (redirectUrl) {
    window.location.href = redirectUrl;
}
```

### 问题2: Profile页面显示空白
**原因**: 可能是CORS问题或API错误

**解决**:
1. 打开浏览器开发者工具 (F12)
2. 查看Console和Network标签
3. 检查API请求是否成功
4. 检查后端是否正常运行

### 问题3: 保存资料失败
**原因**: Token过期或无效

**解决**:
1. 重新登录获取新token
2. 检查localStorage中的access_token是否存在
3. 验证后端API是否正常

### 问题4: 样式显示不正常
**原因**: CSS文件路径错误或缓存问题

**解决**:
1. 硬刷新页面 (Ctrl+F5)
2. 检查navbar.css是否正确加载
3. 清除浏览器缓存
