## 💡 ROI计算说明

### 投资明细
- 人力成本：120万元（10人团队，平均月薪1万元）
- 技术成本：30万元（云服务器、CDN、API调用）
- 市场成本：50万元（线上推广、行业展会、合作伙伴）
- 硬件成本：20万元（ESP32开发板、AR设备等）
- 运营成本：30万元（办公场地、水电、其他支出）

### 收入模型
- 第一年：5,000用户 × 平均ARPU 160元 = 80万元
- 第二年：20,000用户 × 平均ARPU 150元 = 300万元
- 第三年：50,000用户 × 平均ARPU 160元 = 800万元

### ROI计算公式
```
ROI = (总收入 - 总投资) / 总投资 × 100%

第一年: (80 - 250) / 250 × 100% = -68%
第二年: (80+300 - 250) / 250 × 100% = 20%
第三年: (80+300+800 - 250) / 250 × 100% = 220%
```
