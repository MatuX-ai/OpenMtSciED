### 1. 完整K-16教育路径
```
幼儿园(K) → 小学(1-5) → 初中(6-8) → 高中(9-12) → 大学
```

- 专业设备 → 日常用品
- 昂贵材料 → 废旧物品再利用
- 确保教育资源普惠性

### 4. 跨学科融合

- 物理 × 工程 × 数学
- 化学 × 生物 × 环境科学

## 🛠️ 技术实现

### 数据生成脚本
- `generate_openscied_elementary.py`: 小学单元生成器
- `generate_openscied_middle_units.py`: 初中单元生成器
- `generate_openscied_high_school.py`: 高中单元生成器
- `generate_openscied_additional.py`: 补充单元生成器
- `generate_openstax_extended.py`: OpenStax章节扩展生成器
- `generate_gewustan_sample_data.py`: 格物斯坦教程生成器
- `generate_stemcloud_sample_data.py`: stemcloud.cn课程生成器

### 数据合并与导入
- `merge_openscied_data.py`: 多学段数据合并
- `import_to_neo4j_http.py`: Neo4j HTTP API导入器
- `verify_final_data.py`: 数据验证脚本

### 关键技术决策
1. **HTTP API vs Bolt协议**: 由于SSL证书问题，采用HTTP API绕过
2. **示例数据策略**: OpenSciEd网站404时，生成符合规范的示例数据
3. **数据库命名**: Neo4j Aura使用实例ID作为数据库名
4. **批量导入**: 每次处理10个单元，避免超时
