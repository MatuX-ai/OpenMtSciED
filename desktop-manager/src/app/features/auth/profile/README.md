# 个人资料页面 (Profile Component)

## 📍 位置

- **组件文件**: `desktop-manager/src/app/features/auth/profile/profile.component.ts`
- **路由路径**: `/profile`
- **访问权限**: 需要登录 (AuthGuard 保护)

---

## ✨ 功能特性

### 1. 基本信息展示与编辑
- ✅ 用户名
- ✅ 邮箱地址
- ✅ 用户ID
- ✅ 头像（支持上传）
- ✅ 在线编辑模式

### 2. 学习信息
- ✅ 年级水平（小学/初中/高中/大学）
- ✅ 年龄
- ✅ 注册时间

### 3. 学习偏好设置
可选的学习兴趣标签：
- 编程
- 机器人
- 电子制作
- 3D打印
- 人工智能
- 游戏开发
- 科学实验
- 数学建模

### 4. 学习统计（待对接后端）
- 完成课程数
- 学习时长
- 获得证书
- 硬件项目数

## 🔧 技术实现

### 依赖服务
- `AuthService`: 获取当前用户信息
- 使用 RxJS Observable 订阅用户状态变化

### 数据流
```typescript
AuthService.currentUser$ (Observable)
    ↓
ProfileComponent 订阅
    ↓
更新 profile 对象
    ↓
模板渲染
```

### 编辑模式
1. 点击"编辑资料"按钮进入编辑模式
2. 修改表单字段
3. 点击"保存"或"取消"
4. 保存时模拟 API 调用（TODO: 对接真实后端）

## 📝 使用方法

### 在路由中访问
```typescript
// 已配置在 app.routes.ts
{
  path: 'profile',
  canActivate: [AuthGuard],
  loadComponent: () =>
    import('./features/auth/profile/profile.component')
      .then((m) => m.ProfileComponent),
}
```

### 编程式导航
```typescript
import { Router } from '@angular/router';

constructor(private router: Router) {}

goToProfile() {
  this.router.navigate(['/profile']);
}
```

### 模板中的链接
```html
<a routerLink="/profile">个人资料</a>
```
