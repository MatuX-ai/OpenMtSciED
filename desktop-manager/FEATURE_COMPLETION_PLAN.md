#### 1. API 配置功能 ⭐⭐⭐⭐⭐
**重要性：** 应用的核心价值在于 AI 辅助教学，必须配置 API

**现状：** 
- Setup Wizard 只有教师信息步骤
- 缺少 API Provider 配置（OpenAI/本地模型/自定义）
- 无法测试 API 连接

**需要实现：**
```typescript
// 新增 API 配置步骤
interface ApiConfig {
  provider: 'openai' | 'ollama' | 'custom';
  apiKey?: string;
  apiUrl?: string;
  model?: string;
  testConnection: boolean;
}
```

**工作量：** 2-3天

**实施步骤：**
1. 在 Setup Wizard 中添加"步骤4：AI 配置"
2. 创建 API 配置表单组件
3. 实现 API 连接测试功能
4. 保存配置到本地（加密存储）
5. 在 Settings 页面添加 API 配置修改入口

**相关文件：**
```
src/app/features/setup-wizard/
├── api-config-step/          # 新增
│   ├── api-config-step.component.ts
│   └── api-config-step.component.html

├── api-config.service.ts     # 新增

├── api_config.rs             # 新增（加密存储）
```
