## 🎯 功能亮点

### 1. 智能存储估算
系统会根据当前课件数量估算未来存储需求：
- 假设平均每课件 10MB
- 动态计算预估增长
- 帮助用户提前规划存储空间

### 2. 可视化展示
- 进度条直观显示空间使用率
- 清晰的数值格式化（B, KB, MB, GB）
- 分类统计信息

### 3. 安全提示
- 向导中明确告知数据存储位置
- 建议预留足够空间（至少 10GB）
- 强调数据安全性

## 🔧 常见问题

### Q1: 为什么看不到 storage 目录？
**A**: 目录在应用启动时创建。如果您是在开发模式下运行，需要重启 Tauri 应用才能触发目录创建。

### Q2: 如何更改存储路径？
**A**: 当前版本使用固定路径。未来版本将支持自定义存储位置。

### Q3: 存储空间计算准确吗？
**A**: 
- 已用空间：从数据库查询实际文件大小总和（准确）
- 可用空间：通过 fs2 库获取（准确）
- 总空间：简化算法 = 已用 + 可用（近似值）

### Q4: 预估增长是如何计算的？
**A**: 基于历史数据，假设平均每个课件 10MB。公式：`课件数量 × 10MB`

## 🎓 技术架构

### 数据流
```
用户操作
  ↓
Angular Component (Settings/Wizard)
  ↓
TauriService.getStorageInfo()
  ↓
Tauri invoke('get_storage_info')
  ↓
Rust Command (storage.rs)
  ↓
  ├─→ SQLite Query (课件统计)
  ├─→ fs2::available_space() (磁盘空间)
  └─→ walkdir (文件夹大小)
  ↓
返回 StorageInfo 结构体
  ↓
前端渲染展示
```

### 关键技术
- **Rust**: fs2, walkdir, rusqlite
- **Angular**: Material Design, Router, Services
- **Tauri**: Command System, State Management
- **SQLite**: 本地元数据存储
