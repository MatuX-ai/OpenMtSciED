## 🎨 设计特色

### Mac 风格设计语言
- **Apple 品牌色**：#007AFF（经典蓝色）
- **简约配色**：黑白灰层次分明
- **流畅动画**：cubic-bezier(0.4, 0, 0.2, 1)
- **圆润边角**：6px-20px 渐进式圆角
- **细腻阴影**：多层次光影效果

### 响应式设计
- **移动优先**：从小屏到大屏渐进增强
- **断点设置**：640px / 768px / 1024px / 1280px
- **自适应布局**：Grid + Flexbox 组合
- **触摸友好**：大按钮、合适间距

### 交互体验
- **悬停动效**：轻微放大 + 阴影加深
- **点击反馈**：缩放回弹效果
- **平滑过渡**：所有状态变化都有动画
- **加载状态**：表单提交显示进度

## 🔧 技术实现

### Angular 最佳实践
- **独立组件**：使用 `standalone: true`
- **懒加载**：路由级别代码分割
- **类型安全**：完整的 TypeScript 类型定义
- **响应式表单**：表单验证和状态管理
- **依赖注入**：服务层解耦

### 性能优化
- **按需加载**：只加载当前页面需要的组件
- **CSS 内联**：组件级样式封装
- **图片优化**：使用 emoji 代替图标字体
- **减少重绘**：使用 transform 代替 position

### 可访问性
- **语义化 HTML**：正确的标签使用
- **键盘导航**：Tab 键顺序合理
- **焦点管理**：清晰的焦点指示器
- **ARIA 属性**：必要的无障碍标记
